# How to use this content

- Just go throught the slides to construct you first package
- They offer a guide for creating a package that suits CRAN
- The [`package_examples`](package_examples) folder contains the files used to construct and check your package
- If you have the R base and the R Studio installed, you re good to go
- If you cannot access the R terminal inside R studio, just go to the menu `Tools >> Terminal >> New Terminal`
- Have fun !!!
